1. Simple Calculator : 
​ A functional calculator that takes two numbers and an operator (+, -, *, /) from the user to perform basic math.
​Key Concept: switch statements and if-else logic.

​  2. Even or Odd Checker :
​ A program that determines if a given integer is even or odd.
​Key Concept: The Modulo operator (%) and remainder logic.

3. Greater Number Finder :
​ Compares three user-inputted numbers to identify which one is the largest.
​Key Concept: Nested if statements and logical operators (&&).

​4. Swapping Two Numbers (No Third Variable) :
 A logic-based program that swaps the values of two variables using only arithmetic operators (+ and -).
​Key Concept: Value assignment and memory management without a temporary variable.
